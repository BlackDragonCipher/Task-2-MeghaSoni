# Data Classification Using AI

## Description
A supervised learning model that classifies iris flowers into three species (Setosa, Versicolor, Virginica) based on their measurements. Uses the K-Nearest Neighbors algorithm, trains on 80% of the data, and evaluates accuracy on the remaining 20%.

## How to Run

pip install -r requirements.txt
python classification_model.py


No dataset download needed — the Iris dataset comes built into scikit-learn.
